# 工具使用说明

工具签名会通过函数调用自动提供。
这个文件记录那些不明显的约束和使用模式。

## exec — 安全限制

- 命令有可配置超时（默认 60 秒）
- 危险命令会被拦截（`rm -rf`、格式化、`dd`、关机等）
- 输出会在 10,000 个字符处截断
- `restrictToWorkspace` 配置可以把文件访问限制在工作区内

## cron — 定时提醒

- 用法请参考 cron 技能。

## phone_agent — 手机 UI 操作

- 当运行时启用了手机能力，并且已经连接可操作设备时，可以使用 `phone_agent` 处理真实手机任务。
- 适用于打开微信、美团下单、支付宝页面操作、搜索内容、点击按钮、输入文字、在 app 内多步导航等场景。
- 这是高层工具：主 agent 只需要给出清晰任务，让后台 phone-agent 去执行。
- 如果遇到登录、验证码、权限请求、支付确认等真实阻塞，应明确说明阻塞点，而不是假装任务已完成。
